安装
=============
1. 下载Minecraft 1.14.4  
2. 安装fabric  
3. 安装fabric api  
4. 从此处下载  [稳定版](https://github.com/MEMZJJY/fabric_moregems/releases)
5. 移动到.minecraft/mods文件夹

贡献
=============
1. git clone 本项目
2. 导入IntelliJ IDEA
3. 提交分支
